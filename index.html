Using z-index in CSS

What is z-index?

Z-index defines the order of which elements on the page are stacked on top of each other

    By default, elements are shown in this order:
  1. Background and root elements.
  2. Non-positioned elements in order of appearance on the HTML.
  3. Positioned elements in order of appearance on the HTML.
    Div 1: Position static
    Div 2: Position relative
    z-index: 9001
    Div 3: Position relative

Why use z-index?

Keep in mind this is for VISUAL PURPOSES ONLY

You can prevent elements from getting blocked.

How to use z-index

Follow these easy steps:

  • Keep your stacking elements in a single parent element.
  • Make sure they have a position attribute, it could be fixed, relative, or absolute.
  • Elements don't like to stack by default, so remember to move the position around as well.
